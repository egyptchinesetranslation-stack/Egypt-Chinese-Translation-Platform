# Smart Chinese–Arabic Translation & Assistance Platform

## Graduation Project Report

---

# 1. Project Title

**Smart Web Platform for Real-Time Chinese–Arabic Translation and Service Requests**

---

# 2. Abstract

This project presents a web-based platform designed to connect Chinese visitors in Egypt with professional Arabic translators in real time. The system provides three primary services:

* Emergency Assistance
* Tour Guide Services
* Daily Translation Mode

The platform enables Chinese users to submit service requests and communicate instantly with translators through a live chat system. Translators can create professional profiles that include their qualifications, ratings, and service availability.

The platform ensures secure communication, organized pricing, service duration tracking, and post-service rating evaluation.

The system is implemented using **React.js** for the frontend and **Firebase** for backend services including authentication, Firestore database storage, and real-time communication.

---

# 3. Introduction

With the increasing number of Chinese visitors, students, and workers in Egypt, language barriers have become a significant challenge. These barriers become especially critical in situations such as emergencies, tourism activities, and employment communication.

This project aims to solve this issue by developing a **real-time digital bridge** that connects Chinese speakers with Arabic translators through an online platform.

The system allows Chinese users to quickly access professional translators and communicate instantly without facing language limitations.

---

# 4. Problem Statement

Chinese individuals living in or visiting Egypt often face difficulties such as:

* Communicating in hospitals or police stations
* Finding certified tour guides
* Searching for jobs in factories or companies
* Finding trusted translators quickly

Currently, there is **no centralized platform dedicated to Chinese–Arabic emergency translation and service requests**, which creates serious communication obstacles.

---

# 5. Proposed Solution

To address this issue, we propose a **Smart Web-Based Translation & Assistance Platform** that:

* Connects Chinese users with verified translators
* Provides categorized service request options
* Supports real-time chat communication
* Tracks service duration
* Allows rating after service completion
* Displays translator profiles including pricing and qualifications

This system ensures **efficient communication and reliable translator services** for Chinese users in Egypt.

---

# 6. System Users

## 6.1 Chinese User

Chinese users can perform the following actions:

* Register and login to the platform
* Choose a required service category
* Browse available translators
* Send translation requests
* Start real-time chat communication
* Rate translators after service completion

---

## 6.2 Translator

Translators can perform the following actions:

* Register and create professional profiles
* Upload certifications and qualifications
* Set pricing for each service type
* Accept or reject service requests
* Define service duration
* Receive ratings from users

---

# 7. System Features

## 7.1 Emergency Mode

This mode is designed for urgent situations where immediate translation assistance is required.

Features:

* Higher pricing due to urgency
* Instant chat activation
* Priority display in the system

Examples:

* Hospital translation
* Police station assistance
* Government paperwork emergencies

---

## 7.2 Tour Guide Mode

This mode allows Chinese visitors to hire translators as tour guides.

Features:

* Select tour location
* Define service duration (hours or days)
* View translator ratings and experience

---

## 7.3 Daily Translation Mode

This mode is intended for daily communication needs.

Features:

* Flexible hourly booking
* Assistance in daily activities
* Useful in public or professional environments

Examples:

* Shopping assistance
* University registration help
* Business meeting translation
* General daily communication support

---

# 8. System Workflow

1. Chinese user logs into the platform
2. User selects a service type
3. User browses available translators
4. User sends a service request
5. Translator receives a notification
6. Payment is handled outside the platform
7. Translator accepts the request
8. Service duration timer starts
9. After completion, rating option appears
10. Rating is stored in the database

* Role-Based Access Control
